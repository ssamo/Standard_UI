Common Guide

모양 블릿 - 점


  • 순서없는 목록의 모양 블릿스타일 점 목록입니다. 내용이 길어지면 패딩으로 들여쓰기가 적용됩니다.
  • IE는 별도로 블릿높이를 조절합니다. top:크롬+1
