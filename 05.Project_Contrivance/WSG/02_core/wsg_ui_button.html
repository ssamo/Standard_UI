addon-방향 : 어떤것을 올려두다.

레이어로 addon 하는것을 구분하며, 텍스트 옆으로 자연스럽게 붙는것은 구분하지 않는다.

									<button type="button" class="btn btn-basic btn-lg w-200"><i class="ico ico-blog position-l"></i><span>.position-l</span></button>
									<button type="button" class="btn btn-basic btn-lg"><i class="ico ico-blog"></i> <span>Button Left</span></button>
									<button type="button" class="btn btn-basic btn-lg"><span>Button Right</span> <i class="ico ico-blog"></i></button>
									<button type="button" class="btn btn-basic btn-lg w-200"><span>.position-r</span> <i class="ico ico-blog position-r"></i></button>
								
									/* Common - Addon */
									.btn.btn-lg.addon-left {padding-left:50px;}
									.btn.btn-lg.addon-right {padding-right:50px;}
									.btn.btn-lg.addon-left .ico {position:absolute; top:50%; left:10px; transform:translate(0, -50%)}
									.btn.btn-lg.addon-right .ico {position:absolute; top:50%; right:10px; transform:translate(0, -50%)}
