  • GUIDE
  • HTML
  • [this]-wrap : 요소별 최상위 그룹을 의미하며, 디자인 구조의 확장성을 고려하여 선택적으로 사용한다.
  • [this] : 기준이 되는 요소이며, 필요한 경우 부모요소 또는 자식요소를 생성한다.
  • [this]-header : 기준 요소의 제목 그룹으로 사용한다.
  • [this]-body : 기준 요소의 내용 그룹으로 사용한다.
  • Master Layout을 제외한 일반적인 그룹의 구조에 사용한다.
									<div class="요소-wrap">
										<div class="요소">
											<div class="요소-header">
											</div>
											<div class="요소-body">
											</div>
											<div class="요소-footer">
											</div>
										</div>
									</div>
