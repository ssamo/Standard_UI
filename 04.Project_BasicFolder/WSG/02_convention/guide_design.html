Common

네이밍 규약

  • 예약어를 제외한 함축어를 사용하지 않는다.
  • 첫자 또는 풀네임을 사용한다.

유형별 가이드

네이밍 규약
종류 설명 예시
Case1. 구분이 없는 속성 속성의 풀네임을 사용 .display-b {display:block;}
Case2. 구분이 있는 속성 속성의 첫글자를 조합 .va-b {vertical-align:bottom;}
Case3. 구분이 없는 값 값의 첫글자를 사용 .float-l {float:left;}
Case4. 구분이 있는 값 값의 첫글자를 조합 .display-ib {display:inline-block;}

기타

  • 같은 속성의 유형이 다양한 경우 통일 한다.
								/* Scroll */
								.overflow-a {overflow:auto;}
								.overflow-h {overflow:hidden;}
								.overflow-x {overflow-x:scroll;} /* 네이밍 패턴 통일화 */
								.overflow-y {overflow-y:scroll;} /* 네이밍 패턴 통일화 */
