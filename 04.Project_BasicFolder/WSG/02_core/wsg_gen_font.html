  • GUIDE
  • CSS
  • 의미 구분 : 성공, 실패, 경고, 위험, 오류 등의 가이드 컬러
  • 강조 구분 : 기본, 중요한, 강조, 비강조, 기타강조 등의 가이드 컬러
  • 채도 구분 : 사이트에 사용되는 컬러 체계를 정의하며 비공통 컨텐츠에 직접 사용 가능
  • 명암 구분 : 사이트에 사용되는 명암 체계를 정의하며 비공통 컨텐츠에 직접 사용 가능
  • 사용자지정 : 증가, 감소 등의 고유 컬러 지정
