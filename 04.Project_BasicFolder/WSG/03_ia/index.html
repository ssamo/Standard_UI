Information Architecture

    Publish Note

    • 유형 표시방법 - FP : Full Popup, MP : Massge Popup, SP : Select Popup, EP : Etc Popup,
    • 1차 진척관리 - 최초 완료표시 기준으로 날짜를 표시하고, 비고란에 전달사항이 있는경우 표시한다.
    • 2차 결함관리 - 신규/수정/삭제 이슈가 있는 경우 소스코드와 비고란에 동일한 주석문구를 표시한다. (주석 표기규칙 : 날짜 이름 내용 이슈)
    • 3차 최종완료 - 사이트 오픈후에 전체현황을 엑셀로 복사하여 기록을 남기고 비고란은 삭제한다. (최종완료에 대한 산출물 현행화)
